Projects

Electric Kilns

Embedded Systems, Multiphysics modeling, CAD, C++, IoT

Designed and built electric kilns. The system includes custom GUIs, IoT connectivity, and data logging to InfluxDB with visualization via Grafana.

Control System for Gas Kiln

Embedded Systems, C++, PLC, IoT, Data Logging

Designed and built an automated kiln control system, starting with ESP32s and transitioning to a robust Beckhoff PLC. The system includes custom GUIs, IoT connectivity, and data logging to InfluxDB with visualization via Grafana, significantly improving efficiency at a ceramic factory.

1D Neutron Transport Solver

Computational Physics, Matlab, Reactor Physics

Developed a computational solver for 1D neutron transport problems as a part of a Reactor Physics course. The project involved implementing numerical methods to model neutron behavior, showcasing a strong foundation in computational physics and scientific programming.
